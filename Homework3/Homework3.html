Exercise 3

Consider the next algorithm:

xxxxxxxxxx
def Ex2 ( A ):
    D ← build ( A )
    while ¬ is_empty ( D )
        extract_min ( D )
    endwhile
enddef

where A is an array.

Compute the time-complexity of the algorithm when:

  • build, is_empty∈ Θ(1), extract_min∈ Θ(|D|);
  • build∈ Θ(|A|), is_empty∈ Θ(1), extract_min∈ O(log n);

Solution

  • xxxxxxxxxx
    def Ex2 ( A ):
        D ← build ( A ) # Θ(1)
        while ¬ is_empty ( D ) # at each iteration Θ(1)
            extract_min ( D ) # Θ(|D|)
        endwhile # at each iteraton one element is removed -> |D| iterations
    enddef
